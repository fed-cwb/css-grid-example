# CSS Grid

## Flexbox X Grid

---

## Flexbox

- layout flexível
- quaisquer elementos
- uma dimensão
---

## Grid

- layout fixo
- elementos conhecidos
- duas dimensões

---

## Flexbox

```css
container {
  display: flex;
  flex-flow: <flex-direction> <flex-wrap>;
  justify-content: <justify>;
}
```

---

## CSS Grid

```css
container {
  display: grid;
  grid-template-columns: <track-size>;
  grid-template-rows: <track-size>;
  align-items: <align>;
  grid-template-areas: <grid-area-name>;
}

element {
  grid-area: <grid-area-name>;
}
```
